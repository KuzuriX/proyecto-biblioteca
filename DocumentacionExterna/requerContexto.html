Libros

ID: RF-100
Descripción: El sistema debe permitir registrar un libro. Para ello deben incluirse los siguientes datos: el ISBN del libro, título, volumen, editorial, fecha de publicación, autores y descriptores asociados, número de ejemplares y tipo de libro (préstamo o reserva).

Ejemplares

ID: RF-200
Descripción: El sistema debe permitir registrar el ejemplar de un libro. Para ello se debe indica el libro del cuál es un ejemplar, el código, el estado físico (excelente, bueno, malo), fecha de ingreso del ejemplar y condición actual del ejemplar (libre, prestado, reservado, reparación, fuera de uso o perdido).
Estado: Incompleto
Creación: 07/12/2014
Versión: 1.0
ID: RF-201
Descripción: El sistema debe permitir cambiar la condición actual de un ejemplar a algunos de los siguientes valores: libre, prestado, reservado, reparación, fuera de uso o perdido.
Estado: Incompleto
Creación: 07/12/2014
Versión: 1.0
ID: RF-202
Descripción: El sistema debe permitir cambiar el estado físico de un ejemplar a algunos de los siguientes valores: excelente, bueno, malo.
Estado: Incompleto
Creación: 07/12/2014
Versión: 1.0
ID: RF-203
Descripción: El sistema debe permitir consultar un ejemplar por su código o por las características del libro que tiene asociado: ISBN, descriptores, aproximación del título o de autor. Los resultados a mostrar serán: la información del libro, los ejemplares existentes y disponibles.
Estado: Incompleto
Creación: 07/12/2014
Versión: 1.0
