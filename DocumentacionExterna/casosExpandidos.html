Libros

Crear libro

Código: CUFE-100
Nombre: Crear libro
Referencias: Requerimientos asociados
RF-100, RF-101
Descripción: Este caso de uso permite que se cree un libro en el sistema.
Actor(es): Sistema.
Actor Sistema
Este caso de uso inicia cuando el usuario selecciona la opción "Crear libro". El sistema muestra al usuario la pantalla para crear un nuevo libro.
El actor ingresa la información en los campos requeridos y presiona el botón "Guardar". El sistema valida la información ingresada por el actor.
El sistema verifica que el libro no se encuentre ya registrado.
Este caso de uso finaliza cuando el sistema guarda el nuevo libro.
Este caso de uso termina en condición de éxito.

Modificar libro

Código: CUFE-101
Nombre: Modificar libro
Referencias: Requerimientos asociados
RF-102, RF-103, RF-105, RF-106
Descripción: Este caso de uso permite que se modifique la información de un libro registrado en el sistema.
Actor(es): Sistema.
Actor Sistema
Este caso de uso inicia cuando el usuario selecciona un libro registrado en la base de datos y presiona el botón de modificar. El sistema muestra la pantalla para modificar la información de libros.
El actor modifica los datos necesarios y presiona el botón "Guardar". El sistema valida la información ingresada por el actor.
Este caso de uso finaliza cuando el sistema actualiza la información del libro seleccionado.
Este caso de uso termina en condición de éxito.
